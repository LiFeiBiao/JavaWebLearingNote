3. 配置
   * MySQL服务启动
      1. 手动
      2. cmd-->services.msc打开服务的窗口
      3. 使用管理员打开cmd
         * net start MySQL ：启动MySQL的服务。
         * net stop mysql : 关闭mysql的服务

   * MySQL登录
      1. mysql -uroot -p密码
      2. mysql -hip -uroot -p连接目标的密码  
      3. mysql --host=ip --user=root --password=连接目标的密码
   * MySQL退出
      1. exit
      2. quit 
   * MySQL目录结构
      1. MySQL安装目录
         * 配置文件 my.ini 
      2. MySQL数据目录
